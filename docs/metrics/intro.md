## Overview

The Metrics & Analytics API gives you visibility into how your APS integrations are performing — API call volumes, error rates, and latency breakdowns.

## Capabilities

- Query usage metrics by time range and resource.
- Monitor error rates and p99 latency.
- Export data for dashboards and alerting.
