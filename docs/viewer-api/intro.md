## Overview

The Viewer API lets you embed and interact with 3D models directly in your web applications. It supports CAD, BIM, and point cloud formats.

## Capabilities

- Render 3D models in the browser with no plugins required.
- Interact with model data: select, isolate, and measure geometry.
- Extend with custom tools and overlays.
